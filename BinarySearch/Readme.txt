Binary Search Algorithm
    Searching algorithm used in a sorted array by repeatedly dividing the search interval by half.
    Algo:
        i. Divide the search space into two halves by finding the middle index "mid"
        ii. Compare a middle element of the space search with key element
        iii. If the key is found at the middle element, the process is terminated
        iv. If the key is not found a middle element, choose which half will be used as the next search space
            # If the key is smaller than the middle element, then end is moved to mid-1
            # If the key is larger than the middle element, then start is moved to mid+1
        v. The process is continued until the key is found or the total search space is exhausted.
